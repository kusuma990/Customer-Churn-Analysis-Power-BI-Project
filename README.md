# Customer-Churn-Analysis-Power-BI-Project
## Project Overview

This project focuses on analyzing customer churn using the Telco Customer Churn dataset.
The goal is to identify why customers leave, which segments are most at risk, and what business actions can reduce churn.

The dashboard is built using Power BI, including:

- Data cleanin
- Data modeling
- DAX measures
- Interactive visualizations
- Insight generation
